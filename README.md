# quiz_api_app

A Flutter quiz app project scaffolded with the required folder structure and starter files.

## Structure

- `lib/`
  - `main.dart` — App entry point
  - `app_config.dart` — App configuration
  - `models/` — Data models (e.g., `question.dart`)
  - `services/` — Business logic/services (e.g., `trivia_service.dart`)
  - `screens/` — UI screens (e.g., `quiz_screen.dart`, `result_screen.dart`)

## Getting Started

1. Ensure you have Flutter installed: https://docs.flutter.dev/get-started/install
2. Run the app:
   ```sh
   flutter run
   ```
